# Contribuindo

Obrigado por considerar contribuir com o AlgeTab Lock.

## Fluxo recomendado

1. Abra uma issue descrevendo o problema ou melhoria.
2. Crie uma branch curta e objetiva.
3. Faca alteracoes pequenas e focadas.
4. Teste a extensao localmente.
5. Abra um pull request explicando o que mudou.

## Padrao de branches

```text
feat/nome-curto
fix/nome-curto
docs/nome-curto
```

## Padrao de commits

Use mensagens no estilo:

```text
feat: adiciona nova protecao
fix: corrige estado do popup
docs: atualiza readme
```

## Antes de abrir PR

- A extensao carrega sem erro no navegador.
- O popup abre corretamente.
- As funcionalidades alteradas foram testadas.
- O `CHANGELOG.md` foi atualizado quando a mudanca afeta usuario final.
